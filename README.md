# react-roadmap

🧩 Этап 1. Основы JavaScript (3–5 недель)

Прежде чем трогать React, нужно хорошо знать JS.
Что учить:

Переменные, типы данных, функции, условия, циклы — 5–7 дней

Масивы, об’єкти, деструктуризація, spread/rest — 5–7 днів

Callback, Promise, async/await — 5–7 днів

DOM, події, fetch API, JSON — 5–7 днів

📆 Итого: ~1 місяць
🧠 Цель: уметь самостійно писати невеликі скрипти, розуміти, як працює браузер і API.



⚛️ Этап 2. Основи React (3–4 тижні)

Що учити:

Створення проекту (Vite або Create React App) — 1 день

JSX, компоненти, props, state — 7 днів

Рендеринг, події, робота з input — 5 днів

Умовний рендеринг, списки, ключі — 5 днів

Життєвий цикл (useEffect) — 5 днів

📆 Ітого: ~1 місяць
🎯 Цель: вміти створити повноцінний SPA з компонентів.

🧠 Етап 3. Просунутий React (3–5 тижнів)

Що учити:

useState, useEffect, useContext — 7 днів

Кастомні хуки — 5 днів

React Router (навігація) — 5 днів

Управління станом (Context API або Zustand) — 5 днів

Оптимізація (React.memo, useCallback, useMemo) — 7 днів

📆 Ітого: ~1 місяць
⚙️ Цель: писати продуктивні, добре структуровані React-застосунки.


🎨 Етап 4. Стилізація та UI (2–3 тижні)

Що учити:

CSS Modules / Styled Components — 3–4 дні

Tailwind CSS або MUI — 5–7 днів

Адаптивність, dark mode, теми — 5 днів

📆 Ітого: ~2–3 тижні
🎯 Цель: красиво оформлювати свої компоненти.

🌐 Етап 5. Робота з API та сервером (2–3 тижні)

Що учити:

Fetch, Axios — 3 дні

Робота з REST API — 5 днів

React Query або SWR — 5–7 днів

📆 Ітого: ~2–3 тижні
⚙️ Цель: вміти отримувати та відображати дані з бекенду.

🏗️ Етап 6. Інструменти розробника (2–3 тижні)

Що учити:

Git + GitHub — 3–5 днів

NPM, Vite, ESLint, Prettier — 5–7 днів

Деплой (Vercel, Netlify) — 2–3 дні

📆 Ітого: ~2 тижні
🚀 Цель: уміти публікувати свій проєкт у продакшн.


💼 Етап 7. Практика та портфоліо (4–6 тижнів)

Що робити:

Зроби 3–5 проєктів:

ToDo App

Weather App

Blog або Notes App

E-commerce mock

Dashboard з API
📆 Ітого: ~1–1.5 місяця

🎯 Цель: навчитися працювати як розробник, а не просто “вивчити теми”.

| Етап             | Час        |
| ---------------- | ---------- |
| Основи JS        | 4–5 тижнів |
| React базовий    | 3–4 тижні  |
| React просунутий | 4–5 тижнів |
| Стилізація       | 2–3 тижні  |
| API / сервер     | 2–3 тижні  |
| Інструменти      | 2 тижні    |
| Практика         | 4–6 тижнів |

🕒 Разом: ≈ 6–7 місяців стабільного навчання (2–3 години на день).
